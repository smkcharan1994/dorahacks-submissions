# Somnia Smart Yield Bot

**Project:** Somnia Smart Yield Bot  
**Author:** Kalicharan S. M  
**Track:** DeFi Agents (Somnia AI Hackathon)

## Vision
A decentralized AI-powered agent that simulates and distributes yield on user deposits on the Somnia Testnet.

## Deployed Contract
- Contract: `SmartYieldBot`
- Address: `0x7EF2e0048f5bAeDe046f6BF797943daF4ED8CB47`
- Network: Somnia Testnet (or Remix VM if tested locally)

## How to run frontend (local)
1. Serve the `frontend/` folder via a local server:
   - Python: `python -m http.server 3000` (run inside the `frontend` folder)
   - Open: `http://localhost:3000`
2. Connect MetaMask (set to Somnia Testnet).
3. Paste contract address (pre-filled) and interact.
