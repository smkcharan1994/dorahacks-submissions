# 🧠 SomniaInfraAI — Autonomous Infrastructure Agent

## 🌍 Vision
A decentralized AI-powered agent that autonomously monitors, optimizes, and scales Somnia Testnet nodes and smart contracts to ensure continuous performance and cost efficiency.

## 💡 Key Features
- Real-time performance monitoring  
- Self-healing infrastructure logic  
- AI-powered optimization insights  
- Simple and interactive dashboard  

## ⚙️ Tech Stack
- Solidity (Somnia EVM)
- FastAPI + TensorFlow (AI Backend)
- React.js / HTML Dashboard
- MongoDB (for metrics)
- Somnia Testnet

## 🧩 Smart Contract
File: `contracts/InfraAgent.sol`

## 🚀 Deployment
- Deploy the contract via Remix or Hardhat  
- Run backend using: `python main.py`
- Open frontend in browser
